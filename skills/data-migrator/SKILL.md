## 기능 설명
이카운트 ERP 데이터를 신규 시스템으로 안전하게 이관하는 코드를 생성합니다.

## 이관 체크리스트

```
Phase 1: Excel 기초 데이터 이관
  □ 거래처 마스터 (사업자번호 중복 체크 포함)
  □ 제품/품목 마스터
  □ 계정과목 마스터
  □ 도입 기준일 기초잔액

Phase 2: OpenAPI 전표 이관
  □ 이카운트 API 인증 확인
  □ 매출 전표 (연도별 월별 순차 이관)
  □ 매입 전표
  □ 지출결의서
  □ 세금계산서

Phase 3: 정합성 검증
  □ 거래처 수 일치 확인
  □ 전표 금액 합계 일치 확인
  □ 미매핑 계정과목 목록 생성
  □ 최종 검증 보고서 생성

Phase 4: 병행 운영
  □ Cron Job 동기화 스크립트 설정
  □ 정합성 알림 설정 (불일치 시 Slack 알림)
  □ 최종 전환 후 이카운트 해지
```

## 사용 예제

예제 1: Excel 이관 실행
- 입력: "이카운트 거래처 Excel 파일 이관해줘"
- 출력: import_customers() 실행 + 성공/실패/중복 통계 출력

예제 2: OpenAPI 전표 스크래핑
- 입력: "2023~2025년 전표 이관해줘"
- 출력: 월별 스크래핑 진행률 출력 + 완료 보고서 생성

예제 3: 정합성 검증
- 입력: "이관 완료 후 데이터 검증해줘"
- 출력: 이카운트 vs 신규 시스템 수치 비교 보고서
