## 프로젝트 목표
- 이카운트 ERP 완전 대체 + 데이터 완전 내재화
- AI OCR·수요 예측·지능형 알림으로 반복 업무 90% 이상 자동화
- 이카운트 월 사용료 제거

## 기술 스택
| 계층 | 기술 |
|------|------|
| Frontend | React + Tailwind CSS (SPA, 모바일 반응형) |
| Backend | FastAPI (Python) |
| Database | PostgreSQL |
| AI Engine | Google Gemini API |
| 외부 연동 | 이카운트 OpenAPI, 국세청 홈택스, 인터넷뱅킹 API |
| 알림 | 카카오 알림톡 API, Slack Webhook |
